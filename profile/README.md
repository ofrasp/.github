# ofRasp

*openFrameworks with Edit mode. Pi-native. Web-native.*

A distribution of [openFrameworks](https://openframeworks.cc/) that prioritizes composition over inheritance. Aimed at **always-on Edit mode**, **Raspberry Pi**, and the **web**. The toolkit is still openFrameworks: `ofApp`, `ofMain.h`, and the addon ecosystem behave the same way.
